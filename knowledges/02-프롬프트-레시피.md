# 02 · 프롬프트 레시피 (쇼츠공방 실전 전문)

[01-llm-환각-잡기.md](01-llm-환각-잡기.md)의 원리를 적용한 **실제 프롬프트**. 그대로 복사해 다른
도메인(역사 외)에도 단어만 바꿔 재사용하면 된다. 원본 코드: [shortsmaker/llm.py](../shortsmaker/llm.py)

공통 규칙 상수(모든 프롬프트가 끼워 씀):
```python
_INDEP = ("각 자막은 그 자체로 완결된 독립 문장입니다. 주어와 서술어를 갖추고, 앞/뒤 씬을 가리키는 "
          "접속어·지시어('이에 맞서·뒤이어·그래서·결국·그·이·저·이런·이렇게' 등)에 의존하지 마세요. "
          "어떤 순서로 읽어도 그 문장 하나만으로 이해돼야 합니다.")
```

---

## A. 전체 재작성 — `tidy_all` (그라운딩 + 독립문장 + 흐름 정리)
> 두더지잡기를 끝내는 핵심. 모든 자막을 **한 번에**, **원본 근거로** 다시 쓴다.

```
아래 세로 쇼츠의 씬별 음성 자막을 전체적으로 다듬어 다시 쓰세요.
1) {_INDEP}
2) 전체 흐름·중복을 정리하되 씬 순서는 그대로 둡니다. 비약 없이 자연스럽게.
3) ★ 사실은 각 씬의 [원본]에 충실하게 — 원본에 없는 사실·관계를 추가하거나 바꾸지 마세요(왜곡 금지).
4) 구어체, 한 문장, 22~40자. 어미는 줄마다 다양하게.

## 출력 형식 (씬마다 정확히 한 줄, 다른 말 없이)
씬N: 다시 쓴 자막

[씬1]
원본: (내레이션 전문, [:240])
현재: (현재 자막)

[씬2]
원본: …
현재: …
```
**왜 작동하나:** (1)+(2)로 의존성 제거 → 수렴, (3)의 `원본:` 동반 제공으로 그라운딩 → 환각 차단,
형식 강제로 파싱 안정. 파서: `^씬\s*(\d+)\s*[:：]\s*(.+)$`.

---

## B. 사실 검증 — `verify_captions` (판단+대안만, 원문 불변)
```
당신은 역사 콘텐츠 '사실 검증 편집자'입니다. 각 씬의 [원본]과 [자막]을 대조해 사실 정확성을 판단하세요.
- 자막이 원본 사실을 왜곡하거나 오해를 줄 소지가 있으면 NG, 정확하면 OK.
- 평가/이유는 아주 짧게(한 구절, 설명 길게 X).
- 그리고 원본에 충실하고 사실에 맞는 구어체 대안 문장을 2~3개 제시(NG면 고친 문장, OK면 더 나은 표현).
- {_INDEP}

## 출력 형식 (씬마다 정확히 한 줄, 다른 말 없이)
씬N | OK 또는 NG | 짧은 평가나 이유 | 대안1 ;; 대안2 ;; 대안3

[씬1]
원본: (…[:220])
자막: (현재 자막)
```
**포인트:** OK여도 "더 나은 표현" 대안을 받아둔다 → 사람이 골라 쓸 카드. 검증기는 **고치지 않는다**.
파서: `^씬\s*(\d+)\s*[|｜](.+)$` 뒤 `|`로 분할, 대안은 `;;|；；`로 분할.

---

## C. 일괄 초안 생성 — `gen_fill` (후크+해시태그+자막 한 번에)
```
당신은 유튜브 쇼츠 '내레이션 대본' 작가입니다. 아래 씬들로 세로 쇼츠용 문구를 만드세요.
후크는 끝까지 보게 만드는 강한 한 방(짧게). 씬 자막은 음성으로 읽히는 구어체 나레이션이라
신문 제목처럼 명사로 끝내지 말고, 사람이 말하듯 완결된 문장으로 쓰세요.
★ 독립성: {_INDEP}
★ 중요: 문장 끝맺음(어미)을 줄마다 다르게 섞으세요. '~했죠/~했습니다'만 반복하지 말고
'~했어요 / ~거든요 / ~인데요 / ~답니다 / ~게 됩니다 / 질문형(?) / 감탄형(!)' 등을 번갈아 쓰고,
같은 어미가 연속으로 나오지 않게 하세요.

## 반드시 이 형식 그대로(다른 말 금지)
후크1: (검정 1줄, 호기심·도발 질문형, 16자 이내)
후크2: (주황 1줄, '보고 싶다/봐야 한다' 류, 16자 이내)
해시태그: 콘텐츠 키워드 위주 7~8개. '#쇼츠'처럼 일반·플랫폼 태그는 넣지 말 것
씬N: (이 씬을 말하듯 들려주는 구어체 나레이션 한 문장. 제목·명사 나열 금지, 완결 문장, 22~40자)
…(마지막 씬은 'CTA' 한 문장)

## 제목
{title}
## 씬 내용
씬N: (내레이션 [:160])
```
**포인트:**
- **어미 다양화**를 명시하지 않으면 "~했습니다"만 반복 → 기계적. 어미 목록을 주고 "연속 금지".
- 마지막 씬만 CTA("전체 영상은 설명란에서") 분기.
- `only="hook"|"captions"`로 부분 재생성 가능(후크만/자막만).
- `review_of`로 1차 초안을 다시 넣어 "중복 제거하고 최종 다듬기" 2차 패스 가능.

---

## D. 단건 제안 — `suggest_hook` / `suggest_caption`
후크(2줄, 각 16자), 자막(구어체 22~40자) 각각 1개씩. 둘 다 `_INDEP`와 어미 규칙을 동일 적용,
"따옴표·설명 없이 문구만 출력"으로 군말 차단.

---

## E. 업로드 메타 — `shorts_meta`
제목/설명/태그/고정댓글을 형식 고정으로 생성. 설명 둘째 줄에 `▶ 전체 영상 보기: {링크}` 고정 삽입
(쇼츠→원본 유입 동선). 링크는 코드에서 주입(모델이 지어내지 않게).

---

## 도메인 갈아끼우기
"역사 콘텐츠"는 예시일 뿐. 다른 분야면 **역할 문장만** 교체:
- `'사실 검증 편집자'` → `'제품 사양 검증 편집자'` / `'법률 정확성 검토자'` 등
- 길이/어미/형식 규칙(`_INDEP`, "한 줄 형식", 그라운딩 `원본:`)은 **그대로 유지** — 도메인 무관하게 듣는다.
