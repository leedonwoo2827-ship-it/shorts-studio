# knowledges — LLM 콘텐츠 생성 노하우

> 쇼츠공방을 만들며 잡은, **"요약·재생성할 때 끝없이 나오는 환각/표류"** 를 길들인 방법과
> 실제로 쓴 프롬프트를 모아둔 곳. 다른 앱(문서공방·영상공방 등)에도 그대로 가져다 쓰려고 정리한다.

## 무엇이 문제였나
LLM에게 "자막을 다듬어줘 / 흐름을 검토해줘"를 반복하면 두 가지가 계속 터졌다.

1. **사실 왜곡(환각)** — 원본에 없던 사실·관계를 지어냄.
   예: "명나라와 청나라는 조공을 *주고받으며*…" (실제론 주변국이 명·청에 조공).
2. **수렴하지 않음(두더지잡기)** — 한 씬을 고치면 이웃 씬이 어색해지고, 다시 고치면 또 다른 곳이
   어색해져 **무한 반복**. 원인은 "이에 맞서·뒤이어·그래서" 같은 **씬 간 의존 접속어**.
